The F# Open Source Group @ GitHub

The F# Open Source Group is a working group associated with The F# Software Foundation, an organization dedicated to F#, including cross-platform and open-source F#. The F# Open Source Group includes F# (the open source edition) and an affiliated collection of community projects at GitHub.

Help us bring F# to your favorite platforms, and help us empower F# users on all platforms.
